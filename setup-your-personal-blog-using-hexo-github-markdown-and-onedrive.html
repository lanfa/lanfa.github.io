Setup your personal blog using Hexo, Github, Markdown and OneDrive

Published: Tue 10 March 2015
By lanfa

In Setup.

tags: Hexo Github Markdown tech

  • Environment

Windows8.1 64Bit - Installing Git, Node.js and Hexo

GitHub has a Windows client: https://windows.github.com/; Node.js: https://nodejs.org/download/ Sublime Text3: http://www.sublimetext.com/3 OneDrive

  • Github Setting

Create an account; Create a Repository, named "lanfa.github.io";

  • Create a local folder

Open Github Client: Create a new repository under ondrive sync folder, like "blog":

  • Install Hexo

Right click on "blog", select "Open in Git Shell" Type in:

npm install hexo -g
hexo init  
npm install

Note: There might be a warning like "npm warn optional dep failed continuing fsevents hexo", just ignore it.

  • Local Test

    hexo g hexo s

Visit http://localhost:4000 in your webbrowser.

  • Configure and Deploy

The configuration file is a straight forward yaml file located at /_config.yml http://hexo.io/docs/configuration.html

Note: The repo link can be found:

after configuration in Git Shell windows, excute:

hexo generate
hexo deploy

If an error “hexo error deployer not found github” occured, excute:

npm install hexo-deployer-git --save

then, edit _config.yml

deploy:
  type: git

Then, in Git Shell windows, excute:

hexo g
hexo d

After 5-10 mins you should be able to see your blog Visiting http://lanfa.github.io/ in your webbrowser.

  • Add a New Article

    hexo new "postName"

a new file will be automatically generated to hexo\source_posts\postName.md

edit the file: The content can be writen using Markdown
