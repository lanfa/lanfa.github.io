Example - Python to IDL Bridge

Python ?? Comments 25/Nov/2015 Wed

The Python to IDL bridge provides a mechanism for calling IDL code from Python[1].

? How to do it without launching ENVI ?

from idlpy import IDL
  • Get the file, read it as Raster and view it in ENVI
e = IDL.ENVI() # The Envi window will be popped out
File = "C:\\Program Files\\Exelis53\\envi53\\data\\qb_boulder_msi"
raster1 = e.OpenRaster(File)
view1 = e.GetView()
layer1 = view1.CreateLayer(raster1)
  • Raster Convert to Array. In this section, How to put parameters to the GetData() method?
    an alternative way to subset the original data is to process the output 'data', and its type is numpy.ndarray.
#data = raster1.GetData("BANDS=[0], SUB_RECT=[100,449,550,899]") #didn't work
data = raster1.GetData()
  • Array to Raster
new_raster = IDL.ENVIRaster(data)
new_raster.save()
  • Perform MNF Transform
Task = IDL.ENVITask('ForwardMNFTransform') 
Task.INPUT_RASTER = new_raster 
Task.OUTPUT_RASTER_URI = e.GetTemporaryFilename() 
Task.Execute()
  • View the MNF result in ENVI
DataColl = e.Data
DataColl.Add(Task.Output_Raster)
View1 = e.GetView()
Layer1 = View1.CreateLayer(Task.Output_Raster)
