Problem using ENVI

The Seamless Mosaic tool is limited to 10 bands. If you are using hyperspectral imagery, you should create a spectral subset of no more than 10 bands before using the Seamless Mosaic tool.

Solved using GDAL

% run E:\OneDrive\GitHub\python\gdal_merge.py -n 0 -o "D:\salar_subset\hypspex_merge_2.dat" "D:\salar_subset\hypspex_10_2.dat" "D:\salar_subset\hypspex_11.dat"
